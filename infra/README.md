# Infraestructura

Esta carpeta agrupa archivos de soporte para correr el sistema en contenedores.

## Subcarpetas

- `docker/`: dockerfiles y scripts de imagen
- `nginx/`: configuracion de proxy si se usa
- `minio/`: configuraciones de object storage local
- `postgres/`: init scripts y ajustes de base de datos
- `redis/`: soporte de cola y cache
