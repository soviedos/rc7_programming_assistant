# Web App

Aplicación `Next.js` responsable de la experiencia del usuario.

## Responsabilidades actuales

- login con validación amigable
- navegación protegida por sesión
- workspace principal del asistente
- consola administrativa
- cambio de rol desde el perfil de sesión

## Pendiente

- Google SSO real
- integración real con chat y RAG
- CRUD administrativo real

## Pruebas

```bash
docker compose exec web npm test
```
